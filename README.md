# Example JS usage of pigi wallet

## Setup
```
$ npm install
```

## Test it out
```
$ npm start

Heres someones balance! { uni: 0, pigi: 0 }
```

That's it! There's a more full example over in the `@pigi/example-rollup` repo, but this should prove it can be done with NPM.
